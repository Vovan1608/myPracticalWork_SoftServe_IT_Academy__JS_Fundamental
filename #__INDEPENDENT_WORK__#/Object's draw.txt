   класс  Bell
	   состояние:
         _name : String
         _power : Boolean
         _volume : Number (0 - 20)
         _melody : Number (0 - 10)
      поведение:
         getName: String
         on(): void
         off(): void

         _isNumber(Number): Boolean
         getVolume: Number
         volumeUp: void
         volumeDown: void
         getMelody: Number
         setMelody(Number): void

   Часы.
5. Чертеж второго устройства:
   класс Clock
      состояние:
         _name: String
         _power: Boolean
         _hour: Number(1 - 24)
         _minute: Number(1 - 60)
         _melody: Number(1)
         _alarm: Number
      поведение:
         getName: String
         on: void
         off: void
         getPower: Boolean
         _isNumber: Boolean
         getHour: Number
         setHour(Number): void 
         getMinute: Number
         setMinute(Number): void
         getMelody: Number
         getAlram: Number   (если время будильника совпало с текущим временем, 
                             то вкл. будильник с _melody)
         setAlarm(Number): void (использовать методы setHour и setMinute)
